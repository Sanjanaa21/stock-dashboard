# Stock Dashboard Frontend

## Overview

This repository contains the frontend application for the **Stock Dashboard** project. It is built using **React.js** and displays live stock market data by connecting to the backend service via REST APIs and WebSocket (Socket.IO).

---

## Features

- Fetches and displays a list of stocks with their current price and price changes.
- Real-time updates of stock prices using Socket.IO.
- Responsive and user-friendly interface.
- Easy to extend and customize.

---

## Technology Stack

- **React.js** – Frontend framework for building UI components.
- **Axios** – For making HTTP requests to the backend REST API.
- **Socket.IO Client** – For real-time WebSocket communication.
- **CSS** – For styling the application.
